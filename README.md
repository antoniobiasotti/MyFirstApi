# MyFirstAPI

Criação de uma API em C# e .NET na IDE Visual Studio Community 2022.

****

# Explicações

- Pasta Properties: Armazena os Settings.json, que é um arquivo com configurações para fazer o debug (otimizações etc.) 
- Program.cs: Entry point da nossa aplicação
- App Settings: Como se fosse um bloco de notas em json que reune informações privadas e sigilosas da aplicação, como o endereço do banco de dados e credenciais de acesso, por exemplo
